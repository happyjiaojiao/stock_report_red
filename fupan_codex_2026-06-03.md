# 市场总览

2026年6月3日早盘，指数层面延续“科技权重托住结构、个股体感偏弱”的特征。公开开盘信息显示，沪指低开，深成指和创业板指高开，但全市场超3000只个股飘绿，说明指数弹性并没有自然传导成普涨赚钱效应。

板块方向上，通信设备、半导体、家居用品、电子元器件等上涨，煤炭、贵金属、电力、文化传媒、互联网等下跌。结合核心观察票开盘表现看，今天早盘主导力量仍集中在AI硬件/光通信链条，而不是情绪连板全面回暖。

# 主线与轮动

## 最强主线：AI硬件/光通信

昨日复盘确认的AI硬件主线在今天早盘获得延续。核心不是单只票强，而是链条内多个层级同步给出反馈：

1. 中军定锚仍在。
   - `002384 东山精密`高开4.11%，延续昨日总龙头地位。
   - `300502 新易盛`高开1.74%，随后维持强趋势，继续承担趋势核心角色。
   - `601138 工业富联`虽接近平开，但盘中涨幅扩大，说明容量资金并未完全退场。

2. 光通信/光纤分支继续强化。
   - `600487 亨通光电`高开8.28%，盘中涨停。
   - `601869 长飞光纤`高开3.49%，盘中涨停。
   - `600105 永鼎股份`高开2.78%，盘中涨停。

3. 后排分化已经出现。
   - `605589 圣泉集团`低开1.46%，弱于前排。
   - `603256 宏和科技`低开1.91%，说明补涨后排并没有无差别延续。

**交易视角**：主线确认强于昨日预案，但开盘阶段已经走到一致性强化，适合验证主线，不适合无脑追涨停附近前排。

## 轮动支线

### 1. 有色金属

有色仍有承接，`600362 江西铜业`高开2.15%，但今天早盘资金显然更集中在通信设备和AI硬件链。它可以继续作为分流观察方向，但暂时不是早盘第一主攻。

### 2. AI PC

`603890 春秋电子`高开5.23%，说明AI PC高度票仍有辨识度。但从昨日龙头链看，AI PC已经被定义为支线情绪锚。今天早盘即使个股继续强，也不能直接替代AI硬件主线。

### 3. 电力/算电协同

昨日负反馈方向没有修复成主线。`600726 华电能源`低开3.03%，`600162 香江控股`低开0.93%，说明旧方向仍偏弱，不能作为早盘主买方向。

# 早盘风险

1. 市场宽度仍弱，超3000只个股下跌，说明不能把指数或主线强度外推成普涨。
2. AI硬件前排已高开加速，短线追买容易承担一致性回落风险。
3. 后排补涨分化明显，圣泉集团、宏和科技开盘弱于前排，说明主线内部也在筛选。
4. 电力、煤炭、贵金属、互联网等方向开盘偏弱，旧题材负反馈仍会压制风险偏好。

# 结论

今天早盘最明确的结论是：**AI硬件/光通信主线延续确认，但开盘不适合直接追高。**  
更合理的动作是确认主线强度，等待盘中分歧、换手或容量票二次共振，再决定是否参与。
